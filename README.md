# Agent-In-The-Box: 成長型AIエージェント・エンジン

> **「手順を覚える」のではなく、「エージェントと共に歩む」ためのインフラ。**

Agent-In-The-Boxは、単なるプロンプト集ではありません。CursorやClaude CodeといったAIツールの上で稼働し、あなたの思考、決定、日々の変化を自律的に記録・整理し、共に成長していくための**「エージェント・OS」**の雛形です。

## 🌟 コア・コンセプト

1. **Invisible Framework**: あなたが「型」を覚える必要はありません。エージェントが自律的にADR（意思決定記録）を起案し、記憶（state）を整理します。
2. **Self-Monitoring**: エージェントが自分自身の健康状態や整合性をチェックし、常に「賢い」状態を維持します。
3. **Crystallization**: 日々の雑多なチャットやコードから、あなたの「好み」や「知恵」を抽出し、ナレッジとして結晶化します。

## 📦 運用モデル (The Seed Pattern)

このリポジトリは「共通基盤（Seed）」であり、実際の成長（ADRやStateの蓄積）は、**あなた自身のクローズドな環境**で行うことを前提としています。

- **OSSとして**: この基盤をクローンし、最新の「思考プロトコル」を取り入れることができます。
- **プライベートとして**: 蓄積されるあなたの決定や知恵は、あなた自身のローカル、またはプライベートリポジトリとして大切に育ててください。基盤の更新と、個人の成長を分離して管理可能です。

## 🚀 クイックスタート

1. このリポジトリをテンプレートとして使用し、あなたのプロジェクト（または専用の脳リポジトリ）としてクローンします。
2. Cursorの「Agentプロファイル」としてこのリポジトリを指定します。
3. **エージェントを常駐させる**: ターミナルで `python scripts/auto_loop.py` を実行します。エージェントがワークスペースを見守り、気づいたことを `state.md` に書き込みます。

## 🧬 主要機能

- **Auto-Loop Engine (ALE)**: 2時間おきにワークスペースをパトロールし、変更を `state.md` に自動反映。あなたが Cursor を開いたとき、エージェントが最新の情報を把握している状態を作ります。
- **Personality Learning (LoRA Ready)**: `python scripts/collect_training_data.py` を実行することで、あなたのADRからエンジニアとしての「思考の癖」を抽出し、将来的なLoRA学習用データを作成します。

## 📁 ディレクトリ構造

- `.agent/`: エージェントの「脳」と「手足」
  - `brain/`: 現在の状態（state）と短期記憶
  - `persona/`: エージェントの性格とあなたの好み
  - `skills/`: エージェントが実行できる自律スクリプト
- `docs/adr/`: あなたの重要な決断の歴史
- `scripts/`: 自律パトロールや自己診断エンジン
