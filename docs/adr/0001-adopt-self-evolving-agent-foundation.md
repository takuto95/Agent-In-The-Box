# ADR-0001: 成長型AIエージェント基盤の採用

## ステータス
Status: Accepted

## コンテキスト
エンジニアがAIツール（Cursor等）を単なる「道具」としてではなく、自身の思考を拡張する「パートナー」として活用するためには、日々の活動を自律的に記録・整理する仕組みが必要である。しかし、手動での記録は学習コストが高く、継続が難しい。

## 決定
Alter-Ego-Seedに基づいた「自律成長型エージェント基盤」を採用する。

1. **思考プロトコル (CoT)**: すべての対話に思考プロセスを含め、透明性を確保する。
2. **Invisible ADR**: 重要な決定はエージェントが自律的にADRとして起案する。
3. **継続的自己診断**: `test-fitness.py` を用い、エージェント自らがシステムの整合性を保つ。

## 結末
これにより、ユーザーは「型」を覚えるストレスから解放され、自然な対話の中で自身の知識ベースを成長させることが可能になる。
